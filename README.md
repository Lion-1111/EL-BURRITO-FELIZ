# El Patrón — Burritos, Tortas y Tacos Artesanales

Sitio web para **El Patrón**, una taquería artesanal mexicana.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- GSAP (animaciones)
- Lucide React (iconos)

## Desarrollo local

```bash
npm install
npm run dev
```

## Build producción

```bash
npm run build
npm run preview
```
